Privacy Policy

GitOps Cockpit for Argo CD  ·  Effective date: March 16, 2026

Data Collection

GitOps Cockpit for Argo CD does not collect, transmit, or store any personal data. The plugin operates entirely within your local IDE environment.

Credentials

Authentication tokens for Argo CD servers are stored locally on your machine using the operating system's secure credential storage (OS keychain via IntelliJ PasswordSafe). The developer has no access to your credentials.

Network Communication

The plugin communicates only with the Argo CD server URLs that you configure. No data is sent to the plugin developer, Michael Latta Software, or any third party.

Analytics and Tracking

The plugin does not include any analytics, telemetry, tracking, or crash reporting.

Third-Party Services

The plugin does not integrate with or send data to any third-party services other than the Argo CD servers you explicitly configure.

Changes to This Policy

Updates to this policy will be posted at https://msolodev.github.io/gitops-cockpit-for-argocd-site/privacy.

Contact
